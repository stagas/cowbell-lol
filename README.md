# cowbell.lol

Live: https://cowbell.lol

To run locally:

```sh
git clone https://github.com/stagas/cowbell-lol
cd cowbell-lol
npm i

# in one shell
npm run watch

# in another shell
npm run start:web
```

For the dev server you need to create a certificate [as described here](https://github.com/stagas/devito#caching--certificate).
